# Love Maths

A simple and interactive maths game designed to teach the basics of JavaScript and DOM manipulation.

## Features

- **Four Operation Types**: Practice addition, subtraction, multiplication, and division
- **Score Tracking**: Keep track of correct and incorrect answers
- **Random Questions**: Each question is randomly generated for endless practice
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive UI**: Click buttons or press Enter to submit answers

## How to Play

1. Open `index.html` in your web browser
2. Choose an operation type by clicking one of the four colored buttons:
   - Green (+) for Addition
   - Blue (-) for Subtraction
   - Orange (×) for Multiplication
   - Red (÷) for Division
3. Enter your answer in the input box
4. Click "Submit Answer" or press Enter
5. Track your score at the bottom of the page!

## Technical Details

This project demonstrates:
- DOM manipulation with JavaScript
- Event listeners (click and keyboard events)
- Dynamic content updates
- Responsive CSS design
- Math operations and random number generation
